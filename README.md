# cobt-customs

Custom Battletech units in use by Colorado Battletech Denver and Colorado Springs

To access these units in https://mekbay.com/, navigate to Options > Advanced. Then under Additional unit servers add this URL: https://raw.githubusercontent.com/Oddmajo/cobt-customs/refs/heads/main/

## SVG Creation

To create SVGs, use svgmassprinter.bat (available from https://github.com/pavelbraginskiy/mekbay-customs-sample) and follow the directions there as necessary.

Quick reference CLI: `.\bin\svgmassprinter.bat --skip-equipment --units "<input_dir>" -o "<output_dir>"`
